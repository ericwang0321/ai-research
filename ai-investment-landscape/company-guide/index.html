AI 公司全景导读与深研名单

阅读方法

AI 产业的价值并不会只留在模型公司。模型决定能力上限，但应用掌握用户入口，数据决定回答是否可靠，工作流决定产品是否真正进入业务，基础设施则承接训练、推理、部署和治理。长期价值更可能流向能够控制关键入口、沉淀专有数据、形成高切换成本，并把技术优势转化为稳定收入和现金流的公司。

这份导读按六个赛道组织，每家公司用一段话回答五个问题：它做什么，和同类相比有什么不同，向谁收费，为什么客户愿意持续付费，以及现阶段是否值得进入下一轮尽调。判断尽量使用公开可回源的收入、客户、用户、用量、融资和估值信息；没有披露的数据不硬估，也不因为缺失就按零处理。

“优先深研”不是给公司打分，更不是无条件投资建议。入选公司需要同时具备较清楚的控制点、已经出现的商业化证据和仍可讨论的投资路径；估值过高、交易状态变化、收入质量不透明、项目制过重、产品同质化或监管风险，都可能让战略价值很高的公司暂时排在后面。下一步仍要核验财务报表、客户合同、留存、毛利、股权条款和退出路径。

优先深研 10 家

以下名单用于进入下一轮商业、财务、技术与法律尽调，并按当前证据强度排序；它不是无条件投资建议，也不代表现有估值已经提供足够安全边际。后续判断仍应以数据室、合同抽样、客户访谈、单位经济性和股权文件为准。

1. Anthropic

控制点在于把前沿模型能力嵌入代码与高价值知识工作，并以安全品牌、长上下文、Claude Code、企业治理和多云分发形成组合壁垒；这使其更像企业 AI 与开发者工作平台，而不只是聊天产品。公司于 2026-05-28 完成 650 亿美元 Series H，投后估值 9,650 亿美元，并称 5 月初收入 run-rate 已越过 470 亿美元，商业化证据极强但并非审计收入。主要风险是估值容错率极低、训练推理资本强度、云与渠道依赖、低价模型替代及安全监管反噬。值得进入下一轮，重点核验 Claude Code 与 API 收入拆分、毛利、净留存、算力采购承诺和客户集中度。

来源：Anthropic Series H（2026-05-28）；Axios 融资复核（2026-05-28）

2. Moonshot AI / 月之暗面 / Kimi

控制点是消费助手、长上下文、开放权重 MoE 模型、低价 API 与研究、编程、办公 Agent 共用一套产品入口，既争夺用户心智，也争夺开发者分发；护城河来自中文场景、模型效率、迭代速度及模型与应用的闭环。2026-05-07，TechCrunch 援引本轮财务顾问披露，约 20 亿美元融资已完成，估值约 200 亿美元，4 月 ARR 超过 2 亿美元，主要由订阅与 API 拉动；该 ARR 仍非审计口径。风险包括低价推理毛利、算力与出口限制、开放模型削弱 API 独占性、消费端低迁移成本及股东治理。值得深研其付费率、API 毛利、企业合同、留存和股权清洁度。

来源：TechCrunch 融资及 ARR（2026-05-07）；Moonshot AI 产品动态（2026-04-20）

3. AIsphere / 爱诗科技 / PixVerse

控制点是自研视频生成基础模型、消费创作工具、企业 API 与实时交互世界模型同栈，既可用模板和分享获取全球用户，也能把模型能力卖给内容生产团队；单次榜单领先不构成永久壁垒，真正壁垒是生成质量、速度、成本、用户数据与分发飞轮。公司 2026-03-12 宣布完成 Series C，官方称进入独角兽行列；Bloomberg 报道融资额为 3 亿美元，但精确投后估值未披露。官方同期披露超过 1 亿用户、1,600 万月活和累计生成 21 亿条视频；超过 4,000 万美元 ARR 为媒体转述，未见公司新闻稿直接列示。风险是视频模型商品化、算力成本、版权合规和热度向高毛利收入转化。值得核验付费 cohort、API 占比及单条视频毛利。

来源：PixVerse Series C 与运营数据（2026-03-12）；Bloomberg 融资额（2026-03-12）；中国基金报 ARR 转述（2026-03-12）

4. Lovable

控制点是让非工程师从自然语言直接生成、部署并持续修改应用，把 AI coding 的市场从开发者扩到创始人、设计与业务团队；项目分享、生成与修复数据、托管运行时和企业治理可共同形成壁垒。公司称 2026 年 6 月年化收入 run-rate 已超过 5 亿美元、累计项目超 5,000 万个且每周新增 100 万个。最近已完成交易仍是 2025-12-18 的 3.3 亿美元 Series B、估值 66 亿美元；截至 2026-07-08，拟融资 3 亿美元、估值 132 亿美元仍处洽谈，不能当作成交价。风险是生成应用的安全、维护与留存、第三方模型依赖及企业 shadow IT。值得核验净收入留存、企业 ARR、推理毛利和生产环境事故率。

来源：TechCrunch 收入与项目数（2026-06-09）；TechCrunch 新轮洽谈（2026-07-08）；TechCrunch Series B（2025-12-18）

5. Exa

控制点是自建面向 Agent 的网页爬虫、索引、排序、内容抽取和垂直检索，而非转售传统搜索结果；结构化、低延迟、可引用且节省 token 的输出，使其有机会成为 AI 应用访问外部实时信息的基础层。公司于 2026-05-20 完成 2.5 亿美元 Series C，估值 22 亿美元，并披露服务超过 5,000 家公司和 40 万名开发者、跟踪超过 5,000 亿个 URL；收入和 ARR 未披露。护城河来自独立索引、专用检索模型、生产集成与规模成本曲线，风险则是模型平台内置搜索、API 切换成本低、内容授权与爬虫限制以及高 QPS 的基础设施毛利。值得深研其付费查询增长、客户集中、搜索质量留存和单位查询贡献利润。

来源：Exa Series C、客户与索引规模（2026-05-20）

6. Clay

控制点是把 150 多个数据源、瀑布式补全、AI 网页研究、意向信号、CRM 同步与触达动作放进可编排工作台，成为销售和增长 Agent 的数据与行动层，而不是单一联系人数据库；多源市场、工作流模板、社区和深度嵌入共同增加迁移成本。公司确认 2025 年 12 月 ARR 达 1 亿美元、服务 1.4 万客户且企业净收入留存超过 200%。2026-01-28 的最新资本化事件是最高 5,500 万美元员工 tender，对应 50 亿美元估值，不是一级增资；Sacra 估算 2026 年 5 月 ARR 约 1.5 亿美元，非公司披露。风险是数据供应商成本与合规、外呼拥挤、CRM 内置化及估值倍数。值得核验 actions 与数据 credits 的分部毛利、真实付费 cohort 和企业续约。

来源：Clay tender 与公司指标（2026-01-28）；Sacra ARR 估算（2026-06-30，估算截至 2026-05）

7. Sierra

控制点是围绕客服与客户旅程构建可执行、可记忆、跨渠道的 Agent OS，并以结果计费而非仅按席位或 token 收费；这让产品直接对应解决率、转化、留存等业务预算，Agent Data Platform 与大型企业实施能力构成壁垒。公司称 2026-02-06 ARR 超过 1.5 亿美元；2026-05-04 宣布融资 9.5 亿美元、估值超过 150 亿美元，Axios 报道投后估值为 158 亿美元。公司同时披露服务超过 40% 的 Fortune 50，平台处理数十亿次客户交互。主要风险是结果归因争议、语音与长会话成本、定制交付拖累毛利、错误动作损伤客户品牌及既有软件套件捆绑。值得核验合同归因条款、毛利、净留存、人工兜底比例和大客户集中度。

来源：Sierra 融资与客户规模（2026-05-04）；Sierra ARR（2026-02-06）；Axios 交易口径（2026-05-04）

8. Harvey

控制点是把法律研究、合同、尽调、诉讼与合规工作流做成可审计的专业 Agent，并以顶级客户信任、法律工程团队、客户自定义工作流与真实使用数据建立垂直壁垒；在 AI 时代，它争夺的是专业服务操作系统，而非通用问答入口。公司于 2026-03-25 完成 2 亿美元融资，估值 110 亿美元，并披露超过 10 万名律师、1,300 家机构、60 多个国家及 2.5 万个自定义 Agent；公司最近一次明确 ARR 披露为 2025 年 7 月超过 1 亿美元，Sacra 对 2026 年 5 月 3 亿美元 ARR 的数字属于估算。风险是法律错误责任、保密与特权数据、token 成本、客户自建和专业内容平台竞争。值得核验审计 ARR、模型后毛利、错误损失、续约及使用深度。

来源：Harvey 融资与客户数据（2026-03-25）；Harvey 公司 ARR 里程碑（2025-07）；Sacra ARR 估算（估算截至 2026-05）

9. Abridge

控制点是从临床对话进入电子病历，再向护理文档、收入周期、临床决策和支付协同延伸；深度工作流集成、医学语境模型、医院信任与大规模真实对话数据，可能把单点语音记录升级为医疗智能基础层。公司 2026-06-11 披露已在 300 多个医疗系统上线、每年支持超过 1 亿次对话，覆盖机构合计服务 2.5 亿多患者。最近可核验的定价轮是 2025-06-24 的 3 亿美元 Series E，媒体估值 53 亿美元；2026 年 6 月药企战略投资金额及新估值未披露，ARR 亦未披露。主要风险是临床错误、隐私、收入编码审计、电子病历平台捆绑和实施周期。值得核验 ARR、活跃医生渗透率、单院扩张、毛利及安全事件。

来源：Abridge 平台与运营规模（2026-06-11）；WSJ Series E 估值（2025-06-24）；STAT 战略投资（2026-06-11）

10. Databricks

控制点是企业私有数据、计算、权限、血缘和治理：当 Agent 需要安全调用内部数据时，统一目录与 Lakehouse 可成为默认后端；开源生态、大客户嵌入和跨云部署构成复合护城河。公司 2026-02-09 披露收入 run-rate 超 54 亿美元、同比增长超过 65%、AI 产品 run-rate 14 亿美元、过去 12 个月自由现金流为正，并拥有 800 多个年消费 run-rate 超 100 万美元的客户及 2 万多家组织。同期约 50 亿美元股权融资按 1,340 亿美元估值完成，另获约 20 亿美元债务额度；2026-06-08 报道的 1,650 亿至 1,750 亿美元新估值仍属洽谈。风险是云厂商捆绑、平台复杂度、AI 工作负载毛利和高估值。值得核验净留存、FCF 质量、AI 收入增量与客户集中度。

来源：Databricks 财务与融资公告（2026-02-09）；Reuters 新轮洽谈（2026-06-08）

通用 AI

通用 AI 赛道同时覆盖前沿基础模型、高效端侧模型、通用 Agent 与依托第三方模型形成的生产力应用。模型公司的价值主要来自能力领先、开发者生态、企业分发与算力效率；应用公司的价值则取决于能否占据高频入口，把模型能力沉淀为稳定工作流、用户数据和可重复付费。

投资判断不应只看模型榜单或产品热度，而要区分技术领先能否转化为收入、毛利和留存。重点观察 API 与订阅的付费质量、企业合同及部署深度、推理和训练成本、核心团队稳定性与平台依赖；对尚无产品或收入的研究实验室，则应按高方差技术期权而非成熟软件公司审视。

Anthropic

该公司研发前沿大模型，并以 Claude、代码 Agent、API 和企业服务切入高价值知识工作。独特点是安全品牌、长上下文、代码能力及云渠道协同；收入来自个人与团队订阅、企业席位和 API 用量。其商业价值在于成为企业可信模型与开发工具入口，但高估值、算力成本和云平台依赖仍需验证。列入优先深研：产品增长与企业分发较强，需重点核验毛利和留存。

Character.AI

该公司提供角色聊天与 AI 陪伴社区，独特点是沉浸式角色互动、用户生成角色网络和创作者生态，而非单纯追求模型排名。收入可来自会员订阅、广告及技术授权，商业价值在于形成 AI 原生互动娱乐平台。核心风险集中于未成年人保护、心理健康、版权与核心技术团队流失。暂不优先：消费吸引力突出，但安全治理和收入质量的不确定性过高。

DeepSeek / 深度求索

该公司研发推理与通用大模型，以开源权重、低成本训练推理和低价 API 形成行业影响力，独特点是工程效率及对模型价格体系的冲击。收入方向主要是 API 用量与企业服务，商业价值还包括国产模型生态和开发者心智。其客户付费、毛利、治理及跨境合规仍缺少充分披露。暂不优先：技术信号强，但商业化与财务可验证性不足。

Genspark

该公司把搜索、研究、文档、演示、多媒体生成及业务操作整合为 AI 工作空间，独特点是跨工具完成可交付成果，而非只提供答案。收入来自个人订阅、团队席位、企业合同和任务额度，商业价值是成为中小团队的通用生产界面。其壁垒受制于第三方模型、同类平台内置功能和高成本任务的单位经济性。暂不优先：产品覆盖广，但留存、毛利与企业扩张尚待证明。

Granola

该公司提供会议笔记与团队记忆工具，结合用户主动记录、后台转写、会后总结和跨会议检索；不需要机器人账号加入会议、可积累团队上下文并连接外部工具，是其主要特点。收入来自个人免费增值、团队与企业按席位订阅。商业价值在于把会议沉淀为可供外部 Agent 调用的组织数据层，但会议平台原生功能可能压缩空间。暂不优先：会议记录属于高频场景，但独立上下文层的长期护城河仍需验证。

Liquid AI

该公司研发非 Transformer 的高效基础模型，面向端侧、边缘和私有部署，独特点是更低内存占用及与芯片、汽车和设备生态的适配。收入模式以商业授权、OEM、私有化部署和企业合同为主，而非纯按 token 售卖。其商业价值是打开云端巨头较弱的本地推理市场，但架构优势可能被主流模型优化追平，销售兑现周期也较长。暂不优先：路线差异鲜明，收入规模化仍待验证。

Manus / Butterfly Effect

该公司提供能调用多模型、云端电脑、浏览器和外部工具的通用 Agent，可从自然语言任务生成研究、网页、演示及跨系统操作。独特点是工作流广度与交付速度；收入来自任务额度、团队计划、连接器和 API。商业价值在于把聊天升级为可执行自动化，但底层模型依赖、任务可靠性和毛利仍有不确定性。暂不优先：Meta 收购已于 2026 年 4 月被中国监管部门叫停并被要求撤回，当前独立经营状况及后续交易安排仍需核验。

ModelBest / 面壁智能

该公司以 MiniCPM 系列端侧小模型服务手机、电脑、汽车、机器人和行业专网，独特点是能力密度、低内存、本地隐私与国产硬件适配。收入可来自设备授权、私有化部署及行业项目，商业价值在于成为端侧 AI 的模型与系统软件层。当前量产出货、授权价格、收入质量和交付毛利披露有限。暂不优先：技术定位清晰，但需先证明合作已从演示转化为可重复量产收入。

Moonshot AI / 月之暗面 / Kimi

该公司兼具消费端助手、长上下文产品认知、开放模型、API 与研究、编程和办公 Agent，独特点是模型能力与产品入口并重。收入可来自会员订阅、API 用量和企业服务，商业价值在于把用户流量、开发者生态与高价值工作流汇成平台。主要风险是算力成本、付费转化、股权治理及估值兑现。列入优先深研：公司同时拥有基础模型、消费端产品和开放平台，需核验收入质量与单位经济性。

OpenAI

为什么其余公司暂不优先

“暂不优先”主要有六类原因。第一，估值已经提前反映了很长时间的增长，留给新投资者的错误空间很小；第二，收入、客户留存、毛利或用量没有足够公开证据；第三，产品容易被基础模型、云平台或办公套件吸收；第四，商业模式依赖实施、定制或算力补贴，规模扩大未必改善经济性；第五，公司正在被收购、准备上市或缺少清晰的独立一级市场入口；第六，医疗责任、版权、数据合规或跨境监管可能显著改变回报。

这些判断是研究优先级，不是永久结论。若后续出现可验证的收入增长、生产环境留存、单位经济性改善、估值回落、交易状态明确或新的独立分发入口，公司可以重新进入深研名单。相反，优先名单中的公司如果无法解释客户集中度、续约、毛利、现金消耗或股权条款，也应被降级。

结论

AI 时代最值得追踪的公司，不一定拥有参数最多的模型，而是能够持续控制企业或个人完成关键任务时必须经过的节点。模型入口、实时数据、专业工作流和企业数据底座是本轮研究最清楚的四类控制点，但每一种都面临平台上收和价格下降。优先深研的下一步，应把公开叙事拆成合同、留存、毛利、现金消耗、竞争替代和退出条款，用事实判断技术优势最终能否成为股东回报。
