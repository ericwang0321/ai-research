4业务模式

4.1 收入来源

公司通过 Vidu 会员订阅、模型 API 按量收费、企业合同和内容生产解决方案变现。面向物理世界的 Motus/Motubrain 仍处于技术和生态建设阶段，近期商业化主要来自数字内容生成。

4.2 商业价值

Vidu 解决视频生产中的多主体一致性、声画同步、生成速度和可控性问题，可减少短剧、广告、动画及营销内容的制作时间。公司进一步把视频模型延伸到世界行动模型，但该方向能否产生独立收入仍需验证。

5产品与技术

  • Vidu：覆盖文生视频、图生视频、参考生视频和声画同出，并通过 Web 产品及 API 对外服务。
  • Motus/Motubrain：尝试用统一世界模型连接视频理解、生成和机器人行动。
  • TurboDiffusion：公司开源了视频生成加速框架，但最新主力 Vidu S1 模型权重未公开，因此对比表按“闭源”标注。

7竞争格局

生数的直接对手包括 Kling、Runway、AIsphere/PixVerse、Luma 和 Pika。相较只做创作界面的产品，生数拥有自研视频模型和 API；相较拥有大平台分发的可灵，生数的独立性更高，但流量入口和资本资源较弱。世界行动模型让公司拥有更大的技术边界，也会增加研发投入和商业焦点分散风险。

10研究结论

生数科技值得放进视频与世界模型同业对比。下一步应重点确认 Vidu 的收入规模、付费留存和推理毛利，并把内容生成业务与具身世界模型投入分开评估，避免只用用户数或模型榜单判断公司价值。
