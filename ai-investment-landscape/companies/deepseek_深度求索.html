DeepSeek 的独特之处在于它并非先做大规模商业 SaaS,而是先通过开源模型和极低 API 价格建立全球开发者心智。V3/R1 之后,DeepSeek 被视为"AI 的拼多多式价格破坏者": 它迫使 OpenAI、Anthropic、Google、xAI、Mistral 等重新解释训练成本、推理成本与模型护城河。

4业务模式与收费方式

4.1 免费 Chat / App

DeepSeek 官网提供网页版和 App,主打"与 DeepSeek 免费对话"。免费入口的商业意义:

  • 快速扩大用户和品牌认知;
  • 获取真实使用反馈,用于模型迭代和产品优化;
  • 向开发者和企业展示能力,促进 API 转化;
  • 在中国和全球市场形成"低价高能力"心智。

免费产品不是主要短期收入来源,但它是 DeepSeek 打入全球 AI 讨论的入口。2025 年 1 月 DeepSeek App 曾登顶美国 iOS 免费榜并引发全球科技股波动;随后服务遭遇注册限制和攻击报道,说明免费高增长也会带来基础设施压力。

  • cache hit input 几乎免费化,鼓励长上下文和重复 prompt 复用。

4.3 开源权重与企业私有化

DeepSeek 的模型策略以开源权重为核心。R1、V3-0324、R1-0528、V3.2、V4 Preview 等都通过 GitHub/Hugging Face 或官方说明提供权重/技术报告。商业化不一定来自模型许可费,而可能来自:

  • 企业在私有云/本地部署中选择 DeepSeek 模型;
  • 国产云和大厂将 DeepSeek 模型接入 MaaS 平台;
  • API 官方低价服务承担自助开发者使用;
  • 行业客户采购定制化、微调、推理加速、国产芯片适配服务;
  • 国产芯片/云生态把 DeepSeek 作为标杆 workload。

开源同时带来双刃剑: 它极大扩大生态,但也降低直接收费能力,并让竞争者可快速蒸馏、量化、部署和改造模型。

4.4 与云厂商/芯片生态关系

DeepSeek 的模型广泛被中国云厂商和海外开发者平台接入。美国出口管制限制高端 Nvidia GPU 获取,反而强化了 DeepSeek "有限算力效率"叙事。媒体报道中,DeepSeek 也被鼓励或推动适配 Huawei Ascend 等国产芯片,但训练稳定性、互联、软件栈成熟度和推理性能仍需持续验证。

5.2 技术差异化

DeepSeek 的核心技术组合:

  1. MoE 架构: 总参数很大,每 token 激活参数较少,降低推理成本。
  2. MLA / Multi-head Latent Attention: 减少 KV cache 压力,提升长上下文效率。
  3. DeepSeekMoE: 专家路由和负载均衡优化,提高训练/推理效率。
  4. FP8 mixed precision: 在大规模训练中使用低精度优化,降低显存和计算成本。
  5. Multi-token prediction: 提升解码效率和训练信号。
  6. GRPO / 强化学习推理训练: R1/R1-Zero 展示无需大量人工标注也能提升推理能力。
  7. DeepSeek Sparse Attention / DSA: V4 时代主打 1M context 的成本效率。
  8. 开源 distilled models: 让社区用 Qwen/Llama 等基座模型获得小模型推理能力。

5.3 产品效果

DeepSeek-V3 技术报告称 V3 在多个开放模型 benchmark 上领先,并在部分任务上接近闭源模型;R1 官方发布页称性能 on par with OpenAI-o1,且开源 MIT License;V4 官方发布页称 V4-Pro 在世界知识、reasoning、Math/STEM/Coding、agentic coding 上领先开放模型并接近顶级闭源模型。

这些 benchmark 需要谨慎解读。DeepSeek 的真正行业影响不是某个分数,而是:

  • 以极低 API 价格提供接近前沿的可用能力;
  • 开源权重让开发者可私有化部署;
  • 证明训练成本并非只能线性堆算力;
  • 迫使投资者重新评估 Nvidia GPU capex 与模型公司毛利。

5.4 成本叙事

DeepSeek-V3 技术报告披露 full training 需要 2.788M H800 GPU hours,成本约 $5.576M,远低于外界对 frontier model 的认知。这个数字必须正确理解:

  • 它主要指 V3 训练 run 的 GPU 计算成本;
  • 不包括早期实验、团队薪酬、数据工程、失败尝试、基础设施折旧、推理服务、机会成本等完整公司成本;
  • 因此不能简单说 DeepSeek 全部模型只花 $5.6M;
  • 但即便考虑完整成本,DeepSeek 仍展示了显著资本效率。

7单位经济性 / 关键财务逻辑

DeepSeek 的单位经济性是全球模型层的关键研究样本。

7.1 收入端

  • 官方 API 价格极低,靠规模、缓存、MoE efficiency 和低成本推理支撑。
  • 免费 Chat/App 不直接贡献大量收入,但降低获客成本。
  • 开源模型可能减少授权收入,但增加企业私有化、云厂商适配和生态影响力。
  • 如果中国大厂和产业资本参与融资,DeepSeek 可能通过战略合作获得长期调用、私有部署和行业项目。

7.2 成本端

  • V3 官方训练 run 成本约 $5.576M,但完整研发成本高于该数字。
  • 推理成本受 MoE active params、MLA/DSA、cache hit、并发优化影响。
  • 中国高端 GPU 获取受出口管制影响,H800/H20/国产芯片组合会影响训练和推理效率。
  • 若转向大规模免费用户和企业 SLA,需要更大推理集群和运维团队。

7.3 财务逻辑

DeepSeek 的低价 API 可能有三种解释:

  1. 真实成本领先: 技术效率足以支撑低价且有毛利。
  2. 战略补贴获客: 通过低价压低全球价格,以生态和资本支持换份额。
  3. 混合模式: 基础 API 低价,企业私有化/行业方案/战略客户另行收费。

目前公开材料不足以判断哪一种占主导。投资上应重点核验官方 API gross margin、cache hit 比例、企业合同 ARPU、云平台分成和推理集群利用率。

DeepSeek 的护城河:

  • 低成本训练/推理工程能力;
  • 开源权重带来的全球开发者生态;
  • 中国 AGI 叙事和国家级战略关注;

容易被攻击的点:

  • 开源导致模型能力被快速复制;
  • 商业化收入不透明;
  • 高端 GPU 和国产芯片限制;

  • 内容审查影响国际用户使用;
  • 若融资后商业化要求上升,可能冲击长期研究文化。

9团队与背景

梁文锋是 DeepSeek 的核心人物。他来自浙江大学/量化投资背景,创办幻方量化,长期投入自研算力和 AI 研究。DeepSeek 的团队文化被外界描述为研究导向、扁平和重工程效率,与传统大厂 KPI 和 VC 快速商业化路线不同。

团队优势:

  • 量化背景带来的算法、工程、并行计算和硬件优化能力;
  • 对基础模型长期投入,不完全依赖短期融资;
  • 研究发布节奏快,V3/R1/V3.2/V4 均有技术报告和权重;
  • 年轻工程团队愿意做低层系统优化。

团队挑战:

  • 2026 扩招会改变小团队文化;
  • 从研究组织转向商业化公司需要 sales、finance、legal、security、enterprise support;
  • 国际合规和政府关系复杂;
  • 如果战略投资者过多,产品路线和开放策略可能受到影响。

10主要风险

  1. 商业化风险: 开源和免费心智强,但可审计付费收入、企业客户和毛利仍需验证。
  2. 芯片供应风险: 高端 Nvidia GPU 受限,国产芯片训练稳定性和生态成熟度仍有挑战。
  3. 监管与跨境风险: 美国、欧洲、亚洲多地对中国 AI app 的数据安全敏感。
  4. 内容审查风险: 中国监管要求可能影响海外用户信任和模型回答范围。
  5. 开源复制风险: 权重开放降低护城河,竞争者可蒸馏或微调。
  6. 估值风险: $50B+ 估值要求收入和战略价值持续兑现。
  7. 安全风险: 模型可能被用于网络攻击、恶意代码、虚假信息或敏感内容。
  8. 组织扩张风险: 大规模招聘和资本进入可能冲击原有研究效率。
  9. 国际声誉风险: distillation、数据来源、隐私政策等争议可能影响海外采用。
