4业务模式

4.1 C 端订阅和生成额度

PixVerse / 拍我 AI 面向全球和国内创作者，提供文生视频、图生视频、模板、音画同步、视频编辑、多镜头成片等功能。C 端增长逻辑类似 Midjourney/Runway：靠模板传播、生成效果、速度和移动端体验获得用户，再通过订阅或生成额度收费。

4.2 B 端 API 和行业方案

官网明确展示开放平台和 API，场景包括电商营销、互动娱乐、游戏制作、影视制作、动漫动画、广告宣传。API 客户线索包括 Fal、OpenArt、Freepik、AIBY、科大讯飞、思为等，说明它不只是消费产品，也在做模型能力输出。

4.3 Agent 和模板商业化

Agent 创作助手和商用模板库能降低 prompt 门槛，把模型能力包装成可复用工作流。对企业客户来说，价值不是“生成一次视频”，而是把营销素材、短视频、电商素材和品牌内容生产标准化。

7竞争格局

  • 全球：OpenAI Sora、Google Veo、Runway、Pika、Luma、Midjourney 视频能力。
  • 国内：快手可灵、字节即梦、通义万相、腾讯混元视频、生数 Vidu 等。
  • API 生态：Fal、Replicate、Together、云厂商模型市场也可能聚合多家视频模型。

爱诗的竞争优势可能来自速度、模板传播和 API 商业化；最大压力来自大厂模型成本优势和分发入口。

12研究结论

  • Base case：爱诗是中国 AI 视频生成头部创业公司之一，模型和应用双线都值得跟踪。
  • Bull case：模板传播 + API 生态 + 世界模型能力带来高 ARR 和可控推理成本。
  • Bear case：视频生成快速商品化，大厂压低价格，C 端热度难转化为高毛利收入。
